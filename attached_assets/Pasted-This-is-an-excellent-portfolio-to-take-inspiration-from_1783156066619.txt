## Recommended stack

To maximize quality, use:

* **Framework:** Next.js 15 (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion + GSAP + Lenis
* **UI:** shadcn/ui
* **Icons:** Lucide React
* **Forms:** React Hook Form + Zod
* **Email:** EmailJS or Resend
* **Deployment:** Vercel
